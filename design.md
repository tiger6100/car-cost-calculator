# 購車成本計算器 — 介面設計文件

## 應用程式概覽

**目標使用者**：有意從韓國進口車輛至台灣的消費者或代辦業者  
**核心價值**：快速精確估算進口車輛在台灣的總持有成本

---

## 畫面清單

### 1. 計算畫面 (Calculator Screen) — 主頁
**路由**：`/(tabs)/index`

**主要內容**：
- 頂部標題列（App 名稱 + 歷史紀錄按鈕）
- 韓元輸入卡片（KRW 金額輸入、即時換算 TWD）
- 進口稅金卡片（顯示 54.3% 稅率計算結果）
- 手續費卡片（過戶費、運費等固定費用）
- 費用明細總覽卡片（各項費用 + 總計）
- 「計算並儲存」主要操作按鈕

**互動流程**：
1. 使用者輸入韓元金額
2. 即時換算為台幣（使用即時匯率或手動輸入匯率）
3. 自動計算稅金（54.3%）
4. 顯示手續費（固定或可調整）
5. 費用明細總覽即時更新
6. 點擊「計算並儲存」將結果存入紀錄

---

### 2. 匯率畫面 (Exchange Rate Screen)
**路由**：`/(tabs)/exchange`

**主要內容**：
- 當前 KRW/TWD 匯率顯示（大字體）
- 最後更新時間
- 手動輸入自訂匯率
- 匯率趨勢說明（靜態資訊）
- 常用換算表（快速參考）

---

### 3. 紀錄畫面 (History Screen)
**路由**：`/(tabs)/history`

**主要內容**：
- 計算紀錄列表（FlatList）
- 每筆紀錄顯示：日期、車輛韓元價格、總台幣成本
- 點擊紀錄查看詳細費用明細
- 滑動刪除紀錄
- 空狀態提示

---

### 4. 設定畫面 (Settings Screen)
**路由**：`/(tabs)/settings`

**主要內容**：
- 稅率設定（預設 54.3%，可自訂）
- 手續費設定（過戶費、運費等）
- 匯率來源設定（自動/手動）
- 幣別顯示設定
- 關於本應用程式

---

## 關鍵使用者流程

### 主要計算流程
```
啟動 App → 計算畫面
→ 輸入韓元金額（數字鍵盤）
→ 即時顯示台幣換算結果
→ 自動計算稅金（54.3%）
→ 顯示費用明細（稅金 + 手續費）
→ 查看總成本
→ 點擊「儲存」→ 存入紀錄
```

### 查看歷史紀錄流程
```
底部導航 → 紀錄頁
→ 列表顯示所有計算紀錄
→ 點擊紀錄 → 詳細費用明細 Modal
→ 可選擇刪除或重新計算
```

## 元件設計規範

### 輸入卡片
- 圓角：`rounded-2xl`
- 陰影：`shadow-sm`
- 邊框：`border border-border`
- 背景：`bg-surface`
- 內距：`p-5`

### 結果顯示
- 主要數值：`text-3xl font-bold text-primary`
- 輔助標籤：`text-xs text-muted uppercase tracking-wider`

### 主要按鈕
- 背景：`bg-primary`
- 文字：`text-white font-semibold`
- 圓角：`rounded-2xl`
- 高度：`py-4`
- 按壓回饋：縮放 0.97 + 觸覺回饋

### 底部導航
- 4 個分頁：計算、匯率、紀錄、設定
- 圖示：MaterialIcons
- 選中色：Emerald-600

---

## 版面配置原則

- 單手操作優先：重要操作集中在畫面下半部
- 卡片式佈局：每個功能區塊獨立卡片
- 即時計算：所有輸入即時反映結果，無需手動觸發
- 清晰的數字層次：金額使用大字體，標籤使用小字體
